# AI秘書システム PCスペック判定

AI秘書システム用PCの購入候補について、CPU型番、CPU Mark、メモリ、ストレージなどから最低スペック・推奨スペックを満たすか判定する静的SPAです。

CPU型番を入力すると、同梱したCPUデータからCPU Markを自動入力します。PC機種名や販売ページの商品名からCPU候補を探す簡易検索もできます。

## 判定基準

- 最低: CPU Mark 10,000以上 / メモリ16GB以上 / NVMe SSD 512GB以上
- 推奨: CPU Mark 18,000以上 / メモリ32GB以上 / NVMe SSD 1TB以上
- 余裕あり: CPU Mark 25,000以上 / メモリ64GB以上 / NVMe SSD 2TB以上

CPU Markの元データはPassMark CPU Mega Listを参照しています。手動確認する場合はPassMark CPU Listで確認します。
