AI秘書システム PCスペック判定

購入候補PCのCPU Mark、メモリ、ストレージを入力すると、最低スペックと推奨スペックを満たすか判定します。

CPU Markを確認
入力後に判定します

CPU MarkはPassMarkのCPU ListでCPU型番を検索して確認してください。

判定基準

最低

CPU Mark 10,000以上 / メモリ16GB以上 / NVMe SSD 512GB以上。動作可能だが長期運用では余裕が少ない構成。

推奨

CPU Mark 18,000以上 / メモリ32GB以上 / NVMe SSD 1TB以上。AI秘書プロセス、ブラウザ自動操作、連携ツール、ログ保存の常時稼働向け。

余裕あり

CPU Mark 25,000以上 / メモリ64GB以上 / NVMe SSD 2TB以上。複数ユーザー、多数の連携、画像・動画系処理まで見込む構成。
